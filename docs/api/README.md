# API 참조 문서

이 디렉토리에는 LG 브라질 A2A 쇼핑 어시스턴트의 자동 생성된 API 문서가 포함되어 있습니다.

## API 문서 생성

API 문서는, 프로젝트 루트에서 다음 명령어로 생성할 수 있습니다:

```bash
npm run docs
```

이 명령은 JSDoc을 사용하여 소스 코드에서 API 문서를 생성하고 이 디렉토리에 저장합니다.

## 주요 모듈

- **BrowserController**: 브라우저 조작 및 액션 관리
- **IntelligentExtractor**: 지능형 컨텐츠 추출
- **CheckoutAutomation**: 체크아웃 프로세스 자동화
- **CrawlingManager**: 크롤링 작업 관리

자세한 API 설명은 생성된 HTML 문서를 참조하세요.
